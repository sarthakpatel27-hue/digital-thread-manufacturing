# P1 — ECN + EBOM→MBOM Process Pack (Starter)

**Date anchor:** 2025-12-23  
**Goal:** Demonstrate controlled change (ECN), EBOM→MBOM translation, and execution readiness (routing + resources + WI links).

## What problem this solves
Manufacturing teams often lose time and quality due to **revision confusion**, **unclear effectivity**, and **gaps between engineering definition (EBOM) and execution definition (MBOM/routing/WIs)**.

## What you will show (portfolio proof)
- A clear ECN workflow + form + RACI
- A realistic EBOM and MBOM for a simple product
- Routing with operation-level resource mapping (fixtures, gauges, programs, WI IDs)
- Work instructions that reference required resources and records captured
- A change impact matrix and effectivity rules
- Product has outsourced ducts (3), flanges (2), clamps, drill lugs(2)
- Gimbal (2) made in IP cell (OP30) and welded between station 1-2 and 2-3
- Fixture FX1101/1102/1103 (Prep Fixtures) + FX-2001 (assembly/weld/inspect fixture)
